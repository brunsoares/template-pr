Explicação do Template Pull Request

Um template de Pull Request é um documento pré-definido que auxilia desenvolvedores a padronizarem as informações fornecidas ao abrir um novo PR (Pull Request) em um repositório.

  • Facilita a revisão de código.
  • Garante que informações importantes não sejam esquecidas.
  • Ajuda a manter o histórico do projeto organizado.

Exemplo de itens em um template:

  1. Descrição das mudanças
  2. Motivação e contexto
  3. Como testar
  4. Checklist

Utilize sempre o template ao criar um novo Pull Request para melhorar a colaboração no projeto!

Nota: Para utilizar os templates, utilize a pasta .github e personalize com markdown conforme necessário. A criação dos Pull Requests será por meio de componentes novos. Ou seja, utilize o comando ng g c seunovocomponent --skip-tests dentro da pasta /componentes
